# Правила длинных нард — референс проекта backgammon-game

Источник правды для игровых правил. При расхождении с кодом — прав этот файл, код синхронизируется.

При расхождении с SPEC.md — обновляются оба файла.

## Доска и нумерация

- 24 пункта, индексы 1..24.
- Голова белых: пункт **24** (верх-право для белых).
- Голова чёрных: пункт **12** (низ-лево для белых).
- Дом белых: пункты **1–6**.
- Дом чёрных: пункты **13–18**.

## Стартовая позиция

- 15 шашек белых на пункте 24.
- 15 шашек чёрных на пункте 12.
- Остальные пункты пустые.

## Направление движения

Оба игрока движутся **против часовой стрелки**:

- Белые: `24 → 23 → … → 1 → выкид`.
- Чёрные: `12 → 11 → … → 1 → 24 → 23 → … → 13 → выкид`.

## Кубики

- Бросает **сервер** через `crypto/rand`. Никогда не клиент.
- Обычный бросок — 2 пипса.
- Дубль — 4 пипса одинакового значения.
- Если можно использовать только один пипс из двух — обязателен **бо́льший**.
- Если ни один пипс не используется — ход переходит сопернику.

## Постановка на пункт

- На свою или пустую клетку — да.
- На клетку с шашкой противника — **нет** (никогда).
- Битья нет.

## Правило головы

- За один ход с головы можно снять **одну** шашку.
- **Исключение:** только на **первом ходу партии** при выпадении `6:6`, `4:4`, `3:3` — разрешено снять **две** шашки с головы.
- Любой последующий 6:6/4:4/3:3 — снова одна с головы.

## Правило шести

Проверяется **только на конец хода** (после END_TURN), а не после каждого отдельного перемещения.

Запрет действует **одновременно** при двух условиях:
1. У соперника **0 шашек** в его доме (доме соперника):
   - Если блокирует белый — смотрим на пункты 13–18 (дом чёрных).
   - Если блокирует чёрный — смотрим на пункты 1–6 (дом белых).
2. В конце хода у блокирующего получилось **6 или более подряд** занятых пунктов.

Когда хотя бы одна шашка соперника впервые заходит в его дом — ограничение **снимается до конца партии**.

Внутри хода временно держать 6 подряд можно.

Для чёрных учитывать, что блок может пересекать границу 24↔1 (последовательность типа [22, 23, 24, 1, 2, 3]).

## Дом и сброс

- Сбрасывать можно только когда все 15 шашек игрока — в его доме.
- Точный пипс выкидывает шашку с пункта, равного пипсу (для белого: пипс 3 → выкид с пункта 3).
- Переборный пипс (нет шашки на точном пункте) выкидывает с **самого дальнего занятого** пункта дома по своему направлению.
- Перемещение внутри дома по легальному пипсу всегда допустимо.

## Победа и счёт

- **Оин** (1 очко) — проигравший выкинул ≥1 шашку.
- **Марс** (2 очка) — проигравший выкинул 0 шашек И у проигравшего нет шашек в доме победителя и на голове победителя.
- **Кокс** (3 очка) — проигравший выкинул 0 шашек И есть шашка в доме победителя или на голове победителя.

## Право первого хода

- Каждый игрок кидает один кубик. У кого больше — ходит первым.
- Первый ход играется этими же значениями (своё + чужое).
- При равенстве — переброс.

## Инварианты для тестов

- Сумма `|board[i]|` по всем белым шашкам + `borneOff[white]` == 15. То же для чёрных.
- На одном пункте — только один цвет (нельзя «смешать»).
- Только в `Status == WaitingForMove` действия игрока валидны.

## Регионы правил, в которых легко ошибиться

1. **Кто чей дом для правила шести.** Смотрим в дом **соперника** (не свой). Это контринтуитивно.
2. **Исключение для дублей на правиле головы.** Только первый ход партии и только 6:6/4:4/3:3. Не 5:5, не 2:2, не 1:1.
3. **Кокс vs марс.** Различие — есть ли шашка проигравшего в зоне «победитель ещё не выкинул бы оттуда» (дом или голова победителя).
4. **Направление чёрных через границу.** Чёрный с пункта 3 при пипсе 5 идёт не «вперёд за 1», а на пункт 22 (через границу 1→24).
